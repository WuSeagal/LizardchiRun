# 🦎 Lizardchi Run

> 手蜥疊手蜥，一起蜥救蜥！

A Godot 4 puzzle-platformer where 21 lizards take turns trying to rescue their friend stranded in the sky.

---

## 🎮 Gameplay

A lizard is stuck high up on a platform! Your 21 lizards will take turns rushing to the rescue — but after **7 seconds** of energetic movement, each one plays dead and floats in place.

Use their frozen bodies as stepping stones to build a **Living Lizard Ladder** and reach the goal!

### Controls

| Input | Action |
|-------|--------|
| `A` / `D` or `← →` | Move |
| `Space` / `↑` / `W` | Jump |
| `E` | Fast-forward timer to 3s |
| `R` (hold 2s) | Restart |

### Tips
- Position the first few lizards carefully — their frozen bodies form the foundation of your ladder.
- Use `E` to freeze a lizard early when it's in a good spot.
